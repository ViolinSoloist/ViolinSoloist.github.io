O que é HTML

A sigla HTML significa Hyper Text Markup Language. É uma linguagem utilizada para produzir páginas na internet. Com essa linguagem podemos criar páginas na internet. Com essa linguagem, podemos criar páginas que contenham informação nos mais variados formatos: texto, som, imagens e animações.
